연결 설정

이 페이지는 헬로의 GitHub 토큰이 있어야만 동작합니다. 토큰은 이 브라우저에만 저장돼요.

토큰 발급 방법 (1회만, 5분)
1. github.com/settings/personal-access-tokens 접속 → Generate new token
2. Token name: seojae-admin / Expiration: 1년
3. Repository access: Only select repositories → knowledge-archive 선택
4. Permissions → Repository permissions → Contents → Read and write
5. Generate 후 github_pat_… 값을 위에 붙여넣고 저장
